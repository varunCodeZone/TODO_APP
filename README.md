# Todo List App 🚀

## ✨ **Features**
- ✅ **Enter Key Support** - Type task and hit enter !
- 📅 **Date Picker** - Optional date add
- 💾 **Auto Save** - LocalStorage
- 🗑️ **Delete Button** - × click to remove
- 📱 **Mobile Responsive** 
- 🎨 **Modern UI** - Gradient design + animations

Todo-app/
├── todo.html # Main page
├── todo.js # Logic + Enter key + Date
├── todo.css # Beautiful styling




## 🚀 **Quick Start**
```bash 
1. Doubble click todo.html!
2. It will run on your local Browser!
```

## 🎮 How to Use

```bash
Date select (optional)

Type task - "Make dinner", "Gym jaao", etc.

Enter to make task

× button to remove task

Pro Tip: task save in localstorage

📱 Mobile Friendly

Touch-friendly buttons

Perfect scroll + animations
```


## 🔧 **Tech Stack**
```bash
HTML5 - Semantic structure

CSS3 - Flexbox + Gradients + Animations

Vanilla JavaScript - No frameworks!

LocalStorage API - Persistent storage
```
